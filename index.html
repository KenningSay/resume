Проект
Layer by Layer — сайт 3D-печати
Запущен · GitHub Pages

Полноценный бизнес-сайт с несколькими секциями: hero с анимацией частиц (Particles.js), галерея с просмотрщиком изображений, FAQ, форма обратной связи.
